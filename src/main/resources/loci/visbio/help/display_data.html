Displaying a data object

To visualize a data object, select it from the list, then click the "Display >" button. To view the data in an existing display, choose it from the popup menu. To view it in a new display, choose "New 2D display..." or "New 3D display..." and type in a name for the new display. In either case, the data will be added to the specified display and immediately shown onscreen.
