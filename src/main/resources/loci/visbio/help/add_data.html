Adding a data object

To create a data object, click the "Add >" button on the Data panel. A popup menu will appear allowing you to select which data transform to use.

Most data transforms require a parent data object from which to derive their data. (The exceptions are datasets and OME images, which are standalone.) If a parent transform is needed, the currently selected data object will be used. If the selected object is not a valid parent for a data transform for some reason, or if no data object is selected, that transform will be grayed out.

After you choose a name for the new data object, the default settings for that transform will be used to compute the object. However, most transforms have parameters you can adjust by clicking the "Edit" button after the data object has been created.

For more details about the specific data transforms available and their parameters, see the Data transforms topic.
