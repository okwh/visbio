Control panels

VisBio's core features are divided into control panels:

  • The Data panel contains the tools necessary for importing and exporting datasets, and managing derivative data objects.
  • The Displays panel lists all the displays that have been created, with controls for managing them.
  • The Tasks panel shows the operations that VisBio is currently performing, with the ability to halt tasks if desired.
  • The System panel shows technical information about your computer system, with controls to adjust VisBio's memory usage and appearance.
When you first run VisBio, the Data, Displays and Tasks panels are visible in the main window. The System panel can be accessed by choosing "System information" from the "Window" menu.
