Appearance settings

Several controls are available for changing the display's appearance.

Zoom

Use the (zoom in), (zoom out) and (reset zoom) buttons to alter the zoom factor. You can also zoom using Shift + right mouse drag vertically within the display, or with the up and down arrow keys while holding Shift.

Pan

Use the (pan left), (pan right), (pan up) and (pan down) buttons to alter the data's placement within the display. You can also pan using Ctrl + right mouse drag within the display, or with the arrows keys while holding Ctrl.

Rotation

In both 2D and 3D displays, use the (rotate clockwise) and (rotate counterclockwise) buttons to rotate the display, or use the left and right arrow keys while holding Shift.

For 3D displays only, use the (rotate left), (rotate right), (rotate up) and (rotate down) buttons to alter the data's orientation within the display. You can also rotate by dragging the right mouse button within the display, or with the arrow keys while holding Ctrl + Shift.

Aspect ratio

To alter the aspect ratio of the display (i.e., how "square" or "rectangular" it looks), type new values into the "Aspect ratio X" and "Aspect ratio Y" text boxes. For 3D displays, you can change "Z" as well.

Other controls

To toggle whether the display shows a scale along the data axes, check or uncheck the "Scale" box. Use the "Bounding box" checkbox to toggle the white bounding box surrounding the data.

Lastly, VisBio allows recording of movies and capturing of screenshots for any display. To access controls for performing these tasks, click the "Capture" button. A dialog box will appear for creating screenshots and movies. For complete instructions on using this dialog box, see the Capture window topic.
