System panel

The System panel (also known as the "geek panel") contains helpful statistics about your computer such as operating system version, Java Virtual Machine version, and optional installed libraries, as well as a report on VisBio's current memory use. To access it, choose "System information" from the "Window" menu.

The "Dump all" button prints all Java system properties to the Output Console.

The "Clean" button instructs the Java garbage collector to free up any wasted memory that has been temporarily left lying around.

You can also change the amount of memory available to VisBio, the program's graphical "Look & Feel," and, on Windows, the renderer used for visualizing data (OpenGL or Direct3D). See the Changing the memory limit, Changing VisBio's appearance and Changing the renderer topics for more information.
