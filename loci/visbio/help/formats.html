File formats

Commercial instruments produce a diversity of image file types, so we are working to add support for as many of them as possible through the use of our Bio-Formats library. This version of VisBio supports the following formats:

  • TIFF stacks
  • Bio-Rad PIC
  • Metamorph STK
  • Zeiss LSM
  • Zeiss Axiovision (ZVI)
  • Openlab LIFF
  • IPLab
  • Deltavision
  • Olympus Fluoview TIFF
  • Leica (lei + tif)
  • Image Cytometry Standard (ICS)
  • QuickTime movies
  • OME-XML
  • Leica Image File Format (lif)
  • Fluoview FV1000 OIF
  • Fluoview FV1000 OIB
  • Amersham Biosciences GEL
  • Imaris
  • Openlab RAW
  • Gatan Digital Micrograph (DM3)
  • PerkinElmer
  • Image-Pro Sequence
  • Image-Pro Workspace
  • Nikon Electronic Format (NEF)
  • Andor Bio-Imaging Division (ABD) TIFF
  • Encapsulated PostScript (EPS)
  • Flexible Image Transport System (FITS)
  • Digital Imaging and Communication in Medicine (DICOM)
  • Standard image formats: BMP, GIF, JPEG, PGM, PICT, PNG

For a current and complete list of formats, see the Bio-Formats web site.
