File formats

Commercial instruments produce a diversity of image file types, so we are working to add support for as many of them as possible. Currently, VisBio supports several formats, including:

  • TIFF stacks
  • Olympus Fluoview TIFF
  • Bio-Rad PIC
  • IPLab
  • Metamorph STK
  • Openlab LIFF
  • QuickTime movies
  • Zeiss LSM
  • Zeiss ZVI (Axiovision)
  • Single-image formats: JPEG, PNG, GIF, BMP, PICT, DICOM, PGM
Not yet supported (but planned eventually) are Leica, Nikon, Image-Pro, Gatan Digital Micrograph and OME-XML. Note that Leica and Nikon are generally TIFF-compatible, meaning VisBio can import and visualize them, although there is no support for their specific metadata yet.

For a current and complete list of formats, see the VisBio file formats section of the VisBio web site.
