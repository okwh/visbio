File formats

Commercial instruments produce a diversity of image file types, so we are working to add support for as many of them as possible to VisBio. Currently, VisBio supports several formats, including:

  • Bio-Rad PIC
  • Metamorph STK
  • Olympus Fluoview TIFF
  • Openlab LIFF
  • QuickTime movies
  • TIFF stacks
  • Zeiss LSM
  • Zeiss ZVI (Axiovision)
  • Single-image formats: JPEG, GIF, BMP, PICT, DICOM, PGM
Not yet supported (but planned eventually) are IPLab, Leica, Nikon, Image-Pro, Gatan Digital Micrograph and OME-XML.
