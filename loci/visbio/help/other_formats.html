Other formats (JPEG, BMP, etc.)

VisBio supports reading a variety of other single-image formats (not including metadata), including BMP, DICOM, FITS, GIF, JPEG, PGM and PICT.

Support for BMP, DICOM, FITS, GIF, JPEG and PGM is provided by ImageJ (which comes packaged with VisBio).

PICT support requires that QuickTime for Java be installed. If you are running Mac OS X, you probably already have QuickTime installed. If you are running Linux, QuickTime is unfortunately not available. If you do not have QuickTime installed, visit the QuickTime download page. Once QuickTime for Java is installed on your machine, VisBio will be able to import Openlab LIFF files.
