The Capture Dialog Box

The Capture dialog box contains functions for recording of AVI movies of smooth transitions between display positions. To access it, click the "Capture" button on a display's control panel, or on the Displays panel (see the Displays topic for more information).

Creating a list of display positions

To create a movie of a VisBio display in motion, navigate the display to the desired orientation (see the Displays topic for more information), then click the "Add" button. An entry will appear on the list of display positions. Repeat this process for each display position in the chain.

The "Remove" button will eliminate the currently selected entry in the list. The "Up" and "Down" buttons will move the currently selected entry up or down in the list, respectively.

You can click an entry on the list to restore the display to that position. Thus, the list also serves a useful way to capture effective visualization angles, in addition to movie creation.

Creating a movie of transitions

By clicking the "Create movie" button, VisBio will construct an AVI movie consisting of the display being rotated, translated and zoomed between the positions defined in the list. In this way, a movie can be made of the display shifting between any set of orientations.

By default, the transitions start slowly, speed up, then slow down again at each display position, resulting in a smooth animation with emphasis on each of the defined display positions. However, by unchecking the "Emphasize transition at each display position" checkbox, this behavior can be changed so that each transition is completely linear.

Lastly, the speed of the transitions can be controlled with the "Speed" slider.
