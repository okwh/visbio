System panel

The System panel (also known as the "geek panel") contains helpful statistics about your computer such as operating system version, Java Virtual Machine version, and optional installed libraries, as well as a report on VisBio's current memory use.

The "Dump all" button prints all Java system properties to the Output Console.

The "Clean" button instructs the Java garbage collector to free up any wasted memory that has been temporarily left lying around.

You can also change the amount of memory available to VisBio (see the Changing the memory limit topic), as well as the program's graphical "Look & Feel" (see the Changing VisBio's appearance topic).
