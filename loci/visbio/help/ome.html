Uploading to OME

To upload data to an OME (Open Microscopy Environment) database, select the data to be uploaded from the Data panel, click the "Export" button, then choose "Upload to OME" from the menu.

A dialog box titled "Upload to OME" will appear. Specify the address of the OME server (e.g., "cyber.microscopy.wisc.edu"), as well as the username and password of the login to use for upload.

Click the "Ok" button, and the Data status bar will report the status of the upload process as it progresses. If there are any problems during upload, they will be reported in VisBio's "Error console" window, which will appear automatically.
