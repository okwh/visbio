VisBio's Displays

The Displays panel

The Displays panel contains functions managing VisBio's displays.

Creating a display

To create a new 2D display, click the "Add 2D" button. When prompted, enter a name for the new display (or just click OK to use the default name). To create a new 3D display, click the "Add 3D" button instead. Once created, the display will appear in the list to the left.

Removing a display

To delete a display, select it from the list and click the "Remove" button. All parameters for that display (e.g., color settings) will be lost.

Showing a display onscreen

Once you add data to a display (using the Data panel), the display will automatically appear onscreen. However, if the display window or its controls later become hidden, you can bring them back by selecting that display from the list, then clicking the "Show" button.

Display controls

Once a display is visible onscreen, you will see a large panel of controls for manipulating it. Notice the four arrows around the edges of the control panel. Clicking these arrows allows you to change their placement. That is, you can lodge the controls to the left, right, top or bottom of the display itself, or break them off completely into a separate window (just click the arrow corresponding to where the controls are already placed).

Appearance

Several controls are available for changing the display's general appearance.

Use the (zoom in), (zoom out) and (reset zoom) buttons to alter the zoom factor. You can also zoom using SHIFT + right mouse drag vertically within the display.

Use the (pan left), (pan right), (pan up) and (pan down) buttons to alter the data's placement within the display. You can also pan using CTRL + right mouse drag within the display.

For 3D displays only, use the (rotate left), (rotate right), (rotate up) and (rotate down) buttons to alter the data's orientation within the display. You can also rotate by dragging the right mouse button within the display.

To alter the aspect ratio of the display (i.e., how "square" or "rectangular" it looks), type new values into the "Aspect ratio X" and "Aspect ratio Y" text boxes. For 3D displays, you can change "Z" as well.

To toggle whether the display shows a scale along the data axes, check or uncheck the "Scale" box. Use the "Bounding box" checkbox to toggle the white bounding box surrounding the data.

VisBio has flexible color mapping support, for controlling how VisBio colors your data. To adjust the colors, click the "Colors" button. A dialog box will appear for adjusting the color settings. For complete instructions on using this dialog box, see the Colors topic.

Lastly, VisBio allows recording of movies and capturing of screenshots for any display. To access controls for performing these tasks, click the "Capture" button. A dialog box will appear for creating screenshots and movies. For complete instructions on using this dialog box, see the Capture topic.

Data

The "Linked data" drop-down box lists all data objects that are part of this display. To toggle the visibility of a data object, check or uncheck the "Visible" checkbox.

To add another data object to the display, click the "Add <" button, then choose a data object from the popup menu. To remove a data object, select it from the drop-down box, then click the "Remove" button. Lastly, the "Controls" button brings up a set of controls for the selected data object, if it has one.

Position

The "Animate" button cycles through the selected animation axis (specified in the drop-down box to the right of the button), using low-resolution thumbnails. The "FPS" selector allows specification of animation speed in frames per second.

Below the animation controls are a series of sliders corresponding to your data's dimensional axes. These sliders control the currently displayed image (or image stack). Use them judiciously to browse your data's depths!
