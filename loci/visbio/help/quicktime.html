QuickTime movies

VisBio supports reading and writing QuickTime movies (not including metadata), but it requires that QuickTime for Java be installed.

If you are running Mac OS X, you probably already have QuickTime installed. If you are running Linux, QuickTime is unfortunately not available. If you do not have QuickTime installed, visit the QuickTime download page. Once QuickTime for Java is installed on your machine, VisBio will be able to import QuickTime movies. On Windows systems, VisBio will also gain the capability to write QuickTime movies.
